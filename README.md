# Cobra NixOS Config

A declarative NixOS configuration with a macOS-inspired desktop built on Niri.

## What's Inside

**Desktop**
- [Niri](https://github.com/YaLTeR/niri) — scrollable-tiling Wayland compositor
- [Waybar](https://github.com/Alexays/Waybar) — top menu bar + bottom dock (macOS-style)
- [Fuzzel](https://codeberg.org/dnkl/fuzzel) — Spotlight-like app launcher
- [Dunst](https://dunst-project.org/) — notification daemon
- [Kitty](https://sw.kovidgoyal.net/kitty/) — GPU-accelerated terminal with image support

**Theming**
- WhiteSur GTK + icon theme (macOS BigSur clone)
- Apple cursor
- Inter font (SF Pro equivalent)
- Catppuccin Mocha bootloader theme

**Shell**
- Zsh with [zinit](https://github.com/zdharma-continuum/zinit), autosuggestions, syntax highlighting
- [Oh My Posh](https://ohmyposh.dev/) prompt (git + nix-shell aware)
- [zoxide](https://github.com/ajeetdsouza/zoxide), [bat](https://github.com/sharkdp/bat), [lsd](https://github.com/lsd-rs/lsd), [fzf](https://github.com/junegunn/fzf)

**Gaming**
- Steam with GE-Proton
- Gamescope, GameMode, MangoHud
- Lutris, Heroic (Epic/GOG/Amazon)
- Wine + Winetricks

**System**
- NVIDIA open kernel modules (Turing+)
- Btrfs with subvolumes (`@`, `@home`, `@nix`, `@log`, `@snapshots`) via [disko](https://github.com/nix-community/disko)
- Limine bootloader
- PipeWire audio
- Zram swap (zstd, 50% RAM)

## Structure

```
.
├── flake.nix                  # Flake entry (nixpkgs + home-manager + disko)
├── install.sh                 # Auto-installer (run from NixOS ISO)
├── hosts/cobra/
│   ├── configuration.nix      # System config
│   ├── hardware-configuration.nix  # Hardware (auto-generated by installer)
│   └── disk.nix               # Disko partitioning (btrfs subvolumes)
├── modules/
│   ├── nvidia.nix             # NVIDIA driver + Wayland env vars
│   └── gaming.nix             # Steam, Gamescope, MangoHud, Lutris, etc.
├── home/
│   ├── default.nix            # Home Manager entry (CLI tools, packages)
│   ├── zsh.nix                # Zsh + zinit + oh-my-posh
│   ├── niri.nix               # Niri compositor
│   ├── waybar.nix             # Waybar settings (top bar + dock)
│   └── theme.nix              # GTK theme, cursor, fonts, fuzzel, dunst, kitty
└── dotfiles/
    ├── niri/config.kdl        # Niri config
    ├── waybar/style.css       # Waybar CSS
    ├── fuzzel/fuzzel.ini      # Fuzzel config
    ├── dunst/dunstrc          # Dunst config
    ├── kitty/kitty.conf       # Kitty config
    └── oh-my-posh.json        # Prompt theme
```

## Install

Boot the NixOS minimal ISO, then:

```bash
sudo bash <(curl -sL https://raw.githubusercontent.com/rayone121/cobra-nix-config/main/install.sh)
```

The installer will prompt for:
- **Username**
- **Hostname** (default: `cobra`)
- **Timezone** (default: `America/New_York`)
- **Target disk**

It then partitions with disko, generates hardware config, patches all placeholders, and runs `nixos-install`.

## Post-Install

```bash
# Config lives at ~/.config/nixos

# Rebuild after editing config
rebuild

# Update flake inputs
update

# Set a wallpaper + generate theme
wall ~/Pictures/wallpaper.jpg
```

## Key Bindings

| Key | Action |
|---|---|
| `Super + Space` | App launcher (Fuzzel) |
| `Super + Return` | Terminal (Kitty) |
| `Super + Q` | Close window |
| `Super + F` | Maximize column |
| `Super + Shift + F` | Fullscreen |
| `Super + H/J/K/L` | Focus left/down/up/right |
| `Super + Shift + H/J/K/L` | Move window |
| `Super + 1-9` | Switch workspace |
| `Super + Shift + 1-9` | Move to workspace |
| `Super + V` | Center column |
| `Super + R` | Cycle column width preset |
| `Super + E` | File manager |
| `Print` | Screenshot (select area) |
